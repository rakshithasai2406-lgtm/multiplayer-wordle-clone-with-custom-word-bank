# Multiplayer Wordle Clone with Custom Word Banks

Group Project

## Team Members
- Person 1: Frontend UI
- Person 2: Game Logic
- Person 3: Word Bank Management
- Person 4: Admin Panel
- Person 5: Multiplayer Features

## Project Description
A 5-letter word guessing game where players get six attempts to guess the word. Admin users can add custom words, and players can compete using a leaderboard.
